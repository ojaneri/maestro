📱 Integração Maestro com Meta APIEste documento define as regras de negócio, especificações técnicas e tratamento de erros para a integração do sistema Maestro com o ecossistema WhatsApp Business (Meta).⚙️ Configuração RequeridaAo ativar a integração Meta API, o sistema deve obrigatoriamente coletar:ACCESS_TOKEN: Token de acesso da Meta.PHONE_NUMBER_ID: Identificador único do número (Ex padrão: 1001052093085900).🕒 Regra de Ouro: Janela de Atendimento (24h)A interação deve ser otimizada para ser "leve" para a IA, respeitando as restrições da Meta:Conversas Livres: Só são permitidas se o usuário tiver enviado uma mensagem nas últimas 24 horas.Bloqueio de Envio: Se a IA tentar enviar uma mensagem fora dessa janela, o sistema deve recusar, retornar erro e fornecer os templates pré-aprovados.Protocolo de FallbackQuando o prazo de 24h expirar, responda à API/IA com:"Mensagens depois de 24hs do envio do cliente não são aceitas. Por favor, só envie mensagens até 24hs do contato do cliente, ou então use a função para enviar as mensagens pré-aprovadas abaixo:"Templates Exemplo:Template|1|Oi tudo bem {1}? Como vai você na cidade de {2}Template|2|Ola {1}? Como vai você na cidade de {2}Template|3|{1} esta por ai? Como vai você na cidade de {2}, e {3}?Sintaxe de Envio:A IA deve usar exclusivamente a função:template("ID_Template", "var1", "var2", "var3")Exemplo: template("1", "Osvaldo", "Fortaleza")🛠️ Implementação Técnica (index.php)As funções disponíveis para o sistema devem incluir o handler de templates para processar as variáveis dinâmicas e injetá-las no payload JSON da Meta.🔍 Diagnóstico de Erros (Troubleshooting)ErroSintomaCausa ProvávelAção no Código190Invalid OAuth TokenToken expirado ou mal formadoMarcar como não retryável; Logar app ID.131008Parameter missingPayload incompletoValidar via JSON Schema (Zod/Joi) antes do POST.131009Parameter invalidCampo fora das regras (ex: texto longo)Validar limites de caracteres e enums.132000Param count mismatchQtd de variáveis erradaBloquear envio se mismatch com o mapa do template.131026Recipient not reg.Número sem WhatsAppMarcar contato como "unreachable".131047Outside 24h windowTentativa fora da janelaFallback automático para função template.130429Rate limit hitMuitas mensagens por segRetry com backoff exponencial e fila.131053Media errorFalha no download (404)Validar link via HEAD antes de enviar.🚀 Exemplos de Chamada (CURL)Template Simples:Bashcurl -i -X POST \
